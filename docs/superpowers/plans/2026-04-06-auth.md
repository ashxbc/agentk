# AgentK Authentication Pipeline Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Wire BetterAuth (Google OAuth + email/password) into the AgentK Next.js site with a clean centered auth modal, persistent sessions, and a Convex `users` table synced on signup.

**Architecture:** BetterAuth runs as a Next.js App Router catch-all at `/api/auth/[...all]`, backed by Drizzle + Turso (LibSQL) for session/account storage. On every new signup, a BetterAuth after-hook fires a Convex `users.upsert` mutation to store `betterAuthId`, `email`, `username`, and `createdAt`. The Navbar reads the session client-side via `authClient.useSession()` and toggles between a Login button and a pink avatar circle.

**Tech Stack:** Next.js 15 App Router · BetterAuth · Drizzle ORM · Turso (LibSQL) · Convex · Tailwind CSS

- [ ] **Step 2: Create `convex/users.ts`**

```ts
import { mutation } from "./_generated/server";
import { v } from "convex/values";

export const upsert = mutation({
  args: {
    betterAuthId: v.string(),
    email:        v.string(),
    username:     v.string(),
    createdAt:    v.number(),
  },
  handler: async (ctx, args) => {
    const existing = await ctx.db
      .query("users")
      .withIndex("by_better_auth_id", q => q.eq("betterAuthId", args.betterAuthId))
      .unique();
    if (!existing) {
      await ctx.db.insert("users", args);
    }
  },
});
```

## Task 5: BetterAuth server instance

**Files:**
- Create: `lib/auth.ts`

- [ ] **Step 1: Create `lib/auth.ts`**

```ts
import { betterAuth } from "better-auth";
import { drizzleAdapter } from "better-auth/adapters/drizzle";
import { createAuthMiddleware } from "better-auth/api";
import { fetchMutation } from "convex/nextjs";
import { db } from "./db";
import { api } from "@/convex/_generated/api";

export const auth = betterAuth({
  baseURL: process.env.BETTER_AUTH_URL!,
  secret:  process.env.BETTER_AUTH_SECRET!,

  database: drizzleAdapter(db, { provider: "sqlite" }),

  session: {
    cookieCache: { enabled: true, maxAge: 60 * 60 * 24 * 30 }, // 30 days
  },

  emailAndPassword: { enabled: true },

  socialProviders: {
    google: {
      clientId:     process.env.GOOGLE_CLIENT_ID!,
      clientSecret: process.env.GOOGLE_CLIENT_SECRET!,
    },
  },

  hooks: {
    after: createAuthMiddleware(async (ctx) => {
      if (ctx.path.startsWith("/sign-up")) {
        const session = ctx.context.newSession;
        if (session?.user) {
          const { id, email, name } = session.user;
          ctx.context.runInBackground(async () => {
            await fetchMutation(api.users.upsert, {
              betterAuthId: id,
              email:        email ?? "",
              username:     name ?? (email ?? "").split("@")[0] ?? "user",
              createdAt:    Date.now(),
            });
          });
        }
      }
    }),
  },
});
```

  const [email, setEmail]       = useState("");
  const [username, setUsername] = useState("");

    setView("login");
    setEmail("");
    setUsername("");
    setPassword("");
    setError("");

    reset();
    onClose();

    setError("");
    if (!email || !password) { setError("Email and password are required."); return; }
    setLoading(true);
    const { error: err } = await authClient.signIn.email({ email, password });
    setLoading(false);
    if (err) { setError(err.message ?? "Invalid credentials."); return; }
    handleClose();
  }

    setError("");
    if (!email || !email.includes("@")) { setError("Enter a valid email."); return; }
    setView("signup-username");
  }

  function handleSignupUsernameNext() {
    setError("");
    const re = /^[a-zA-Z0-9_]{3,20}$/;
    if (!re.test(username)) { setError("Username must be 3–20 chars: letters, numbers, underscore."); return; }
    setView("signup-password");
  }

  async function handleSignupSubmit() {
    setError("");
    if (password.length < 8) { setError("Password must be at least 8 characters."); return; }
    setLoading(true);
    const { error: err } = await authClient.signUp.email({ email, password, name: username });
    setLoading(false);
    if (err) { setError(err.message ?? "Signup failed."); return; }
    handleClose();
  }

  return (
    <div
      className="fixed inset-0 z-50 flex items-center justify-center bg-black/30"
      onClick={handleClose}
    >
      <div
        className="w-full max-w-[400px] bg-white rounded-2xl border border-black/[0.08] p-8 mx-4"
        onClick={e => e.stopPropagation()}
      >

        {/* ── LOGIN ── */}
        {view === "login" && (
          <>
            <h2 className="text-xl font-extrabold tracking-tight text-[#191918] mb-1">Welcome back</h2>
            <p className="text-sm text-[#B2A28C] font-medium mb-6">Sign in to your AgentK account</p>

            <button
              onClick={handleGoogle}
              className="w-full flex items-center justify-center gap-3 py-2.5 rounded-xl border border-black/10 bg-white text-sm font-semibold text-[#191918] hover:bg-[#fafafa] transition-colors mb-4"

              {GOOGLE_ICON} Continue with Google

              <div className="flex-1 h-px bg-black/8" />

                  onChange={e => setPassword(e.target.value)}
